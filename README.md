# Complexity-Final-Project
Python and R code files used for the analysis in Childhood Trauma Predicts Increased Adult Depression Symptom Interconnectivity in Population-Level Networks (Complexity: Can it be Simplified? final project)

## File Descriptions
- group_creation_final.ipynb — Jupyter Notebook for the merging of waves, cleaning of the data, creation of groups, and exporting relevant data of the two analysed groups to .csv
- bootstrapped_subsampling_analysis_final.R — R code for the bootstrapped subsampling method comparing networks of 2,000 random samples
- matched_n_NCT_final.R — R code for the Network Comparison Test of the high trauma group and one random subsample of the no trauma group

**Note:** The Add Health dataset is not provided here, as I am not authorized to share that data. Therefore, the code cannot be run online. However, the version used for the analysis can be downloaded from the ICPSR website: https://www.icpsr.umich.edu/web/ICPSR/studies/21600#.
